Potato Disease Prediction using CNN

Technologies: Python, TensorFlow, Keras, OpenCV
A deep learning-based image classification system designed to detect diseases in potato leaves from captured images. Using a Convolutional Neural Network (CNN) architecture, the model classifies leaves into categories such as healthy, early blight, and late blight. The dataset is preprocessed with OpenCV for image resizing and augmentation, while TensorFlow & Keras handle model training and evaluation. This system can assist farmers in early detection, reducing crop loss and improving yield through timely treatment.
